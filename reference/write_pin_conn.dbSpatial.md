# Write a dbSpatial object to a pins board

Write a dbSpatial object to a pins board

## Usage

``` r
# S3 method for class 'dbSpatial'
write_pin_conn(x, board, name, ...)
```

## Arguments

- x:

  A dbSpatial object

- board:

  A pins board object

- name:

  Name for the pin

- ...:

  Additional arguments passed to pins::pin_write

## Value

Invisibly returns the input object
